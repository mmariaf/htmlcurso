A Estrutura Básica do HTML

Um documento HTML válido precisa seguir obrigatoriamente a estrutura composta pelas tags html, head e body e a instrução !DOCTYPE. Esta estrutura está informada em uma documentação que descreve todos os detalhes do HTML, no caso as tags e atributos, e como os navegadores devem considerar e interpretar estas tags, esta documentação é chamada de "especificação do HTML", e através do que está declarado nela que é possível entender se um documento HTML válido. Um documento HTML inválido é carregado pelo navegador, porém em um "modo de compatibilidade", vamos entender melhor sobre isto logo mais.

Abaixo, vamos conhecer em detalhes cada uma das tags estruturais obrigatórias:

Tag HTML

A tag <html> é uma das tags fundamentais do HTML (HyperText Markup Language), que é a linguagem de marcação padrão para a criação de páginas da web. Essa tag é usada para definir o início e o fim de um documento HTML.

A tag <html> geralmente é a primeira tag em um documento HTML e é usada para delimitar todo o conteúdo da página. Todas as outras tags HTML devem ser colocadas dentro da tag <html>.

A tag <html> também pode conter um atributo "lang" que especifica o idioma principal do conteúdo da página. Isso é importante para garantir que os navegadores e outros sistemas de software possam exibir o conteúdo corretamente para usuários de diferentes idiomas.


Tag Head

A tag <head> do HTML é usada para definir informações sobre a página, como seu título, links para folhas de estilo CSS e meta-informações que descrevem a página para os motores de busca e outros sistemas que acessam o conteúdo da web. O conteúdo dentro da tag <head> não é visível para os usuários que acessam a página, mas é usado por navegadores da web e outros sistemas para determinar como exibir e acessar o conteúdo da página.

A tag <head> é uma tag de contêiner e é geralmente colocada logo após a tag <html> de abertura e antes da tag <body> que contém o conteúdo da página. O conteúdo dentro da tag <head> não é visível para os usuários que acessam a página, mas é usado por navegadores da web e outros sistemas para determinar como exibir e acessar o conteúdo da página.

Alguns dos elementos mais comuns que podem ser incluídos dentro da tag <head< incluem:

  • <title>: define o título da página, que é exibido na barra de título do navegador.
  • <meta>: define metadados que descrevem a página, como a descrição da página, as palavras-chave usadas na página e informações sobre o autor da página.
  • <link>: define links para arquivos externos, como folhas de estilo CSS e arquivos de ícone.
  • <script>: define scripts JavaScript que são executados na página.

Tag Body

A tag <body> é uma das principais tags do HTML (HyperText Markup Language), que é usada para definir o conteúdo visível da página da web. Todos os elementos da página que devem ser exibidos no navegador da web devem estar contidos dentro da tag <body>. Isso pode incluir texto, imagens, vídeos, formulários e muitos outros tipos de conteúdo. A tag <body> também pode conter atributos que controlam a aparência e o comportamento do conteúdo da página, como a cor de fundo da página, a cor do texto e as funções de rolagem.


Declaração !DOCTYPE

A instrução <!DOCTYPE> é uma declaração que informa ao navegador da web qual versão do HTML está sendo usada no documento. É geralmente a primeira linha do documento HTML e é seguida pela tag <html>. O <!DOCTYPE> não é uma tag HTML, mas sim uma declaração que ajuda o navegador a interpretar o código HTML corretamente. A declaração <!DOCTYPE> é importante porque diferentes versões do HTML têm sintaxe e recursos diferentes, portanto, o navegador precisa saber qual versão está sendo usada para renderizar a página corretamente. Além disso, algumas versões mais antigas do Internet Explorer podem ter problemas de compatibilidade com a versão mais recente do HTML, por isso é importante especificar a declaração <!DOCTYPE> para garantir a renderização correta da página em todos os navegadores.



Atributos

Atributos do HTML são recursos adicionais que podem ser adicionados às tags HTML para fornecer informações extras sobre como o conteúdo deve ser exibido ou comportar-se na página. Cada tag HTML pode ter vários atributos, e esses atributos variam dependendo da tag. Por exemplo, a tag <img> tem o atributo "src" para especificar a localização da imagem, enquanto a tag <a> tem o atributo "href" para especificar o URL do link.

Existem alguns atributos HTML genéricos que podem ser adicionados a qualquer tag HTML, independentemente do seu tipo ou função específica. Aqui estão alguns exemplos de atributos genéricos do HTML:

  • class - permite que você defina um ou mais nomes de classe CSS para uma tag, que pode ser usada para aplicar estilos específicos a essa tag.
  • id - permite que você atribua um identificador exclusivo a uma tag HTML, o que pode ser útil para identificar essa tag para fins de estilização ou manipulação com JavaScript.
  • style - permite que você defina estilos CSS diretamente em uma tag HTML, em vez de usar uma folha de estilo externa ou incorporada.
  • title - permite que você adicione um título descritivo a uma tag HTML, que pode ser exibido como uma dica de ferramenta quando o usuário passa o cursor sobre a tag.
  • data-* - permite que você crie seus próprios atributos personalizados começando com o prefixo "data-", que podem ser usados para armazenar informações adicionais sobre um elemento para uso em JavaScript ou CSS.


Tags Específicas para Textos

Existem algumas tags HTML específicas para o texto que são usadas para fornecer formatação e estruturação adicionais ao conteúdo textual em uma página da web. Aqui estão alguns exemplos:


Tag p: Criando Parágrafos

A tag <p> é usada para definir parágrafos de texto. Qualquer texto dentro dessa tag será tratado como um bloco de parágrafo, com um espaçamento vertical adicional acima e abaixo do texto.

Exemplo:

Lorem, ipsum dolor sit amet consectetur adipisicing elit. Quo, ex! Amet iste doloribus, perferendis quas dolorem asperiores labore ab? Id magnam temporibus nihil architecto! Sunt perferendis quam impedit repellendus qui.


Tag h1 - h6: Criando Cabeçalhos

Essas tags são usadas para definir os cabeçalhos de uma página. h1 é o maior e mais importante, e h6 é o menor e menos importante. Eles podem ser usados para criar uma hierarquia clara de seções e subseções em uma página.

Exemplo:

Cabeçalho 1

Cabeçalho 2

Cabeçalho 3

Cabeçalho 4

Cabeçalho 5
Cabeçalho 6

Tag blockquote: Criando Citações

A tag blockquote é uma tag HTML usada para citar um trecho de texto de outra fonte em um documento HTML. Ela é frequentemente usada para citar texto de outras publicações, como artigos de notícias, livros e revistas. O navegador normalmente exibe o texto dentro de uma tag blockquote com uma formatação de recuo, para destacá-lo do restante do texto. Além disso, muitas vezes a tag cite é usada dentro de uma tag blockquote para fornecer informações sobre a fonte do texto citado, como o autor e a publicação original. Isso pode ajudar os leitores a verificar a precisão da citação e a encontrar a fonte original do texto, se desejado.

Exemplo:
Lorem ipsum dolor sit amet consectetur adipisicing elit. Numquam aut porro beatae deleniti quo modi quam rem consequatur id, quasi eos, inventore nihil a ad obcaecati, dicta laborum ipsa fugiat!

Tag strong: Negrito

A tag strong é uma tag HTML que é usada para enfatizar o texto de uma maneira mais forte do que a tag em. Ela indica que o texto deve ser exibido em negrito. Embora a tag strong seja frequentemente usada para enfatizar palavras ou frases importantes em um texto, é importante notar que a tag strong não deve ser usada apenas para tornar o texto mais visível ou atraente visualmente. A tag strong deve ser usada apenas para destacar o significado ou importância do texto. Por exemplo, ela pode ser usada para enfatizar uma palavra-chave em uma frase ou para destacar uma frase que é particularmente importante para o significado geral de um texto.

Exemplo:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Architecto suscipit iste id maiores, facere dolor accusamus quasi, perspiciatis fugiat voluptates minima, dolorem veniam nostrum possimus. Deserunt praesentium cum ut porro.


Tag em: Ênfase

A tag 'em' é usada para enfatizar o texto em um documento HTML. Geralmente, os navegadores exibem o texto dentro da tag 'em' em itálico, embora isso possa ser modificado por meio de CSS. A ênfase do texto dentro da tag 'em' é importante para a compreensão do significado da frase ou sentença. Por exemplo, em um texto que discute a importância da saúde mental, a palavra "saúde" pode ser enfatizada usando a tag 'em' para destacar sua relevância para o tema em questão. Além de fornecer ênfase, a tag 'em' também é usada por tecnologias assistivas, como leitores de tela, para ajudar a identificar o conteúdo mais importante em uma página da web.

Exemplo:

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Rem vitae, voluptas voluptatibus hic, dignissimos culpa neque numquam distinctio similique quisquam pariatur? Nostrum rem corrupti fugiat et distinctio quia repudiandae fuga?


Tag u: Sublinhado

A tag 'u' é usada para criar texto sublinhado em um documento HTML. Embora seja menos comum do que era no passado, ainda pode ser usado para fornecer ênfase ou destacar informações importantes. Por exemplo, em um documento de pesquisa, uma palavra ou frase que precise ser destacada pode ser sublinhada usando a tag 'u'. No entanto, é importante notar que o uso excessivo de texto sublinhado pode tornar a leitura do conteúdo mais difícil e pode prejudicar a acessibilidade para pessoas com deficiência visual. Como alternativa, pode-se usar a tag 'em' para enfatizar o texto, ou a formatação CSS, como negrito ou itálico, para destacar informações importantes.

Exemplo:

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Doloribus, corporis esse. Sint provident tenetur earum eos nemo doloremque consequuntur beatae mollitia dolore aut necessitatibus repellat, ipsam unde quam nam corporis?


Tag I: Itálico

A tag i é uma tag HTML que é usada para enfatizar o texto de uma maneira diferente da tag strong. Ela indica que o texto deve ser exibido em itálico. Embora a tag i seja frequentemente usada para enfatizar palavras ou frases em um texto, é importante notar que ela não deve ser usada apenas para tornar o texto mais visível ou atraente visualmente. A tag i deve ser usada apenas para destacar o significado ou importância do texto de uma maneira sutil. Por exemplo, ela pode ser usada para indicar um termo técnico em um texto ou para enfatizar uma palavra que é usada em um contexto incomum.

Exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Minima accusantium incidunt aliquam officiis saepe vel voluptas, ab quisquam amet voluptatem dolor magnam autem eaque fugiat nobis accusamus nemo! A, sit?


Tag Mark: Marca-Texto

A tag 'mark' é usada para destacar um trecho de texto em um documento HTML. Ela permite que um trecho de texto seja destacado com uma cor de fundo diferente do restante do texto, tornando-o mais fácil de ser notado. A tag 'mark' é útil para indicar termos importantes em um texto, como palavras-chave ou frases relevantes, ou para destacar citações importantes em um artigo. É importante lembrar que o uso excessivo de 'mark' pode tornar o conteúdo confuso e prejudicar a legibilidade e acessibilidade do texto, portanto, deve ser usado com moderação.

Exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Ipsam obcaecati iure ipsa, voluptate, sint unde corrupti consequuntur, sequi iusto harum dolores! Maxime suscipit nulla hic unde ipsam, voluptatibus modi esse.


Tag sup e Tag sub: Texto na Parte Superior e Inferior da Linha

As tags HTML 'sup' e 'sub' são usadas para criar texto sobrescrito e subscrito, respectivamente.

A tag 'sup' é usada para criar texto sobrescrito. Isso é comumente usado para exibir expoentes em fórmulas matemáticas ou químicas. Por exemplo, a equação "2^3" seria escrita em HTML como "23" e seria exibida como "23" com o "3" sendo sobrescrito.

A tag 'sub', por outro lado, é usada para criar texto subscrito. Isso é comumente usado para exibir índices em fórmulas matemáticas ou químicas, ou para exibir números de série em química e biologia. Por exemplo, o número de série de uma molécula de água, H2O, seria escrito em HTML como "H2O" e seria exibido com o "2" sendo subscrito.

Exemplo:

Loremipsum dolor sit amet consectetur adipisicing elit. Cumque inventore provident soluta porro iure voluptatibus fuga dolorem! Voluptas enim perferendis minus fuga, impedit laudantium quasi laborum necessitatibus pariatur similique tempora!



Tags para Listas

Aqui estão algumas das principais tags HTML para listas:

  • 'ul': é a tag para uma lista não ordenada (unordered list). É usada para criar uma lista de itens que não seguem uma ordem específica. Cada item da lista é marcado com a tag 'li' (list item).
  • 'ol': é a tag para uma lista ordenada (ordered list). É usada para criar uma lista de itens que seguem uma ordem específica, geralmente numerada. Cada item da lista é marcado com a tag 'li'.
  • 'dl': é a tag para uma lista de definição (definition list). É usada para criar uma lista de termos e suas definições correspondentes. Cada termo é marcado com a tag 'dt' (definition term), e cada definição é marcada com a tag 'dd' (definition description).


Tags para Links e seus Atributos

A tag 'a' é uma das tags mais importantes do HTML, pois é usada para criar links entre páginas da web ou para outras partes do mesmo documento. O elemento 'a' é usado com o atributo href para especificar o URL do documento ou recurso que o link levará o usuário quando clicado.

Além do atributo href, a tag 'a' também pode ter outros atributos, como:

  • 'target': especifica onde o recurso será aberto quando o link for clicado.
    • _blank: Abre o link em uma nova janela/tab do navegador.
    • _self: Abre o link na mesma janela/tab do navegador.
    • _parent: Abre o link no frame "pai", caso exista.
    • _top: Abre o link no topo da janela/tab do navegador, substituindo todas as frames, caso existam.
  • 'title': fornece informações adicionais sobre o link, como uma descrição ou título.
  • 'rel': especifica o relacionamento entre o documento atual e o documento vinculado pelo link. Por exemplo, "nofollow" informa aos mecanismos de pesquisa para não seguir o link.
